# 🤖 AI Resume Analyzer

## 📌 Project Description

The **AI Resume Analyzer** is a Python-based application that evaluates a user's resume against a job description using Natural Language Processing (NLP).
It provides a **resume score, skill analysis, feedback, and an ideal resume example** to help users improve their resumes for better job opportunities.

---

## 🎯 Objectives

* Analyze resumes using AI techniques
* Compare resumes with job requirements
* Identify missing skills and gaps
* Provide actionable suggestions
* Generate an **ideal resume (100% match)**

---

## 🚀 Features

* 📥 User input-based resume creation
* 🧠 Resume analysis using TF-IDF and cosine similarity
* 📊 Resume scoring system
* 🔍 Skill detection and missing skill identification
* 💡 Detailed feedback and improvement suggestions
* 🌟 Ideal resume generation aligned with job requirements

---

## 🛠️ Technologies Used

* Python
* Scikit-learn (TF-IDF, Cosine Similarity)
* Basic NLP techniques

---

## ⚙️ Installation

Install required libraries:

```
pip install scikit-learn
```

---

## ▶️ How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run the program:

```
python main.py
```

---

## 📥 Input Required

The user needs to enter:

* Name
* Skills (comma separated)
* Projects
* Soft Skills

---

## 📤 Output

The system generates:

### 📊 Resume Analysis

* Resume Score (%)
* Detected Skills

### 🧠 Feedback

* Strength of resume
* Missing skills
* Suggestions for improvement

### 🌟 Ideal Resume

* A **perfectly optimized resume example**
* Includes required skills, strong action verbs, and impactful project descriptions
